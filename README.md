SLOT GAME

О проекте:
- Простая слот-игра на TypeScript с PixiJS
- Используется MVVM + машина состояний (FSM)

Компоненты:
1. Модель (SlotModel)
- Хранит баланс, текущую ставку, последний выигрыш
- Снимает ставку и добавляет выигрыш
- Список доступных ставок и переключение между ними

2. ViewModel (SlotViewModel)
- Связывает модель с интерфейсом
- Реагирует на кнопки SPIN, AUTO, STOP, BET
- Управляет стадиями игры и авто-спинами
- Эмитирует события для UI

3. Сцена (SlotScene)
- Рисует рилы, кнопки, баланс и выигрыш
- Подписана на события ViewModel и FSM
- Запускает/останавливает рилы и анимацию выигрыша

4. FSM (машина состояний)
- Стадии: IDLE, SPINNING, SHOWING_WIN
- Не даёт нарушить последовательность действий

5. Сервис моков (SlotMockService)
- Генерирует случайные спины
- Считает выигрыш по центральной линии
- Возвращает матрицу символов и сумму выигрыша

Поток данных:
- Игрок нажимает кнопку (SPIN / AUTO / STOP / BET)
- UI вызывает метод ViewModel
- ViewModel проверяет условия, меняет стадию и шлёт события
- Сцена подписана на события и обновляет UI
- После завершения спина модель обновляется, UI тоже

Как запустить:
- npm install
- npm run dev
